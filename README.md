# 遊戲廳 arcade
## 一個可以與真人遊玩各種遊戲的平台
包括但不限於21點、抽鬼牌及麻將等經典遊戲 <br>
## 21點遊戲說明
<!-- 21點遊玩截圖 -->
開場莊家發牌，每人兩張明牌(大家都可以看的到的)，莊家一張明牌，一張暗牌(等大家都停牌之後才能看的)<br>
本遊戲使用撲克牌遊玩，A可以作為11或1，由玩家自行決定，J、Q、K皆為10，其餘卡牌則不變<br>
發完牌先由最右邊的玩家開始，玩家可以選擇要牌、停牌、分牌及加倍<br>
如果玩家認為自己卡牌點數總和不接近21點，可以選擇要牌<br>
請注意，如果要牌之後點數超過21點，即爆牌，這回合出局<br>
如果認為繼續要牌有爆牌的風險，可以選擇停牌，將輪到左邊的玩家<br>
如果玩家有兩張相同的牌，可以選擇分牌，即擁有兩手牌(須加倍賭注)<br>
如果玩家認為勝率大，可以選擇加倍，即加倍賭注，但會自動多要一張牌<br>
如果10秒內無任何操系統會自動選擇停牌<br>
<small>註: 莊家點數17點以下會自動要牌，如點數為17點或以上則停牌</small>
